System and Installation

Works with any browser, just extract and open index.html file under your browser file menu.

Safe Center Application

Use the application by simply interacting with the navigation panel and clicking on a center, or entering a center in the search box. This application illustrates the model-view-view model pattern and how data from different sources can be mediated and presented for local use.

Components

The folder stucture follows:
  • css - containing the style sheet,
  • js - with libraries including Knockout java script and the main application file scriptmap5.js
    • Changes can be made to these files.

Use Cases

The application was developed assuming that there will be many markers. Implemented use cases are:

  • Check items in the nav bar.
  • Search for the location of a center, spell accurately.
