System and Installation

Works with any browser, just extract and open index5.html file under your browser file menu.

Concept Map Application

This application illustrates how data from different sources can be mediated and presented for local actions.
Use the application by simply interacting with the navigation panel and clikcing 'Submit'. Refresh to start a new session.

Components

The folder stucture follows:
  • css - containing the style sheet,
  • js - with libraries including Knockout java script and the main application file scriptmap5.js
    • Changes can be made to these files.

Use Cases

The application was developed assuming that there will be many markers. Hence inactive markers are made invisible to reduce clutter. This is also a best practice. Implemented use cases are:

  • Check one or more items in the nav bar.
  • Search for the location of a center, spell accurately.
