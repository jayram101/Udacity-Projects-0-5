Evade Bugs Game

The game is to move the Girl from the base and cross the river, capturing points by moving over the gems along the way. This game is written in HTML5, Java Script and uses the animaition frame

Getting started

Download the zip from here and extract the files.

To run

Game works within any web browser. From your web browser open the index.html file. The game will start.

To play

Use arrow keys to move the sprite across the grid. Position over gems to earn points. Avoid the bugs as you will end up back where you started. Move through the river and relaod to start again.

Directory stucture

The main directory has the main HTML index file, the subdirectories and files are:

  • css: style.css which governs the look of the page in which the game display,
  • images: with different images you can use for the game,
  • js: which has three files app.js, resources.js, engine.js. The engine.js is the main loop and uses the animation frame; resource.js loads images; and app.js contains most of the applicaiton logic which can be modified. In line documentation explains this further.
